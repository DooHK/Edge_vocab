# 📖 영어 단어장

영어 단어를 번역하고 저장해서 퀴즈로 복습하는 웹 앱입니다.

## 기능
- 🔍 **번역**: 영어 단어·문장 → 한국어 (MyMemory 무료 API)
- 📚 **단어장**: 번역한 단어 저장, 검색, 삭제
- ✏️ **퀴즈**: 저장된 단어로 랜덤 퀴즈
- 📥 **CSV 내보내기/가져오기**: 단어 백업 및 공유
- 📲 **PWA 지원**: 앱처럼 설치 가능

## 사용 기술
- 순수 HTML + CSS + JavaScript (프레임워크 없음)
- [MyMemory Translation API](https://mymemory.translated.net/) (무료)
- localStorage (브라우저 단어 저장)
- Service Worker (오프라인 지원)
